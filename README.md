# GYscan - Go语言综合渗透测试工具

## 项目简介

GYscan 是一款基于 Go 语言开发的综合渗透测试工具，专注于资产探测、漏洞检测和安全验证。该工具由 BiliBili 博主「弈秋啊」开发，提供了丰富的安全测试功能，适用于授权的安全评估和渗透测试场景。

## 功能特性

### 核心功能模块

1. **网络扫描**
   - 端口扫描与服务识别
   - 子域名枚举
   - FTP 服务扫描与爆破
   - SSH 服务扫描与爆破

2. **Web 安全**
   - 目录扫描
   - XSS 漏洞检测
   - CSRF 漏洞检测
   - Web 指纹识别
   - WAF 检测

3. **数据库安全**
   - 数据库弱口令爆破
   - 支持 MySQL、PostgreSQL、MongoDB 等主流数据库

4. **系统安全**
   - 配置审计
   - 弱口令检测
   - 风险评估

5. **其他工具**
   - 加密解密工具
   - 生成密码字典
   - 网络数据包分析

## 安装与使用

### 环境要求

- Go 1.24+ 开发环境
- 支持 Windows/Linux/macOS 操作系统

### 安装步骤

```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd GYscan/Client

# 编译项目
go build -o GYscan.exe
```

### 快速开始

```bash
# 查看帮助信息
./GYscan help

# 执行端口扫描
./GYscan nmap -t target.com

# 执行目录扫描
./GYscan dirscan -u http://target.com
```
